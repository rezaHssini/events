# Planner Studio (`@event/planner-mobile`)

Android shell (Capacitor) for organizer tools. Web UI lives in `@event/planner-ui`.

```bash
# Dev UI
npm run dev -w @event/planner-ui

# Signed APK
npm run apk:planner:release

# Upload
DEPLOY_HOST=root@65.109.221.245 npm run deploy:planner-apk
```

Download: `http://<host>:3040/downloads/planner.apk`
